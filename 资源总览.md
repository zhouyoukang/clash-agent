## 本项目整合清单

| 资源 | 来源 | 本地位置 | 更新方式 |
|------|------|----------|----------|
| Mihomo 内核 | MetaCubeX/mihomo | `clash-meta.exe` | `update-resources.ps1 -Only core` |
| MetaCubeXD 面板 | MetaCubeX/metacubexd | `ui/` (108文件) | `update-resources.ps1 -Only ui` |
| GeoIP 数据库 | Loyalsoldier/v2ray-rules-dat | `geodata/geoip.dat` (19MB) | `update-resources.ps1 -Only geo` |
| GeoSite 域名库 | Loyalsoldier/v2ray-rules-dat | `geodata/geosite.dat` (10MB) | 同上 |
| MaxMind GeoIP2 | MetaCubeX/meta-rules-dat | `geodata/Country.mmdb` (8MB) | 同上 |
| 远程规则集 (8个) | Loyalsoldier/clash-rules | `clash-config.yaml` rule-providers | 引擎自动每24h |
| 自研管理面板 | 本项目 | `vpn-manager.py` (988行) | 手动 |
| CLI 管理 | 本项目 | `proxy-manager.ps1` | 手动 |
| 配置生成器 | 本项目 | `gen_config.py` | 手动 |

## 一键全量更新

```powershell
.\update-resources.ps1           # 更新面板+地理数据+内核
.\update-resources.ps1 -Only ui  # 仅更新面板
.\update-resources.ps1 -Only geo # 仅更新地理数据
.\update-resources.ps1 -Only core # 仅更新内核
python gen_config.py             # 重新生成配置
```
