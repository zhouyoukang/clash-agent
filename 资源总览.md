## 一键更新命令

```powershell
.\update-resources.ps1              # 全部 (面板+地理+内核+CF测速)
.\update-resources.ps1 -Only ui     # 仅 MetaCubeXD 面板
.\update-resources.ps1 -Only yacd   # 仅 Yacd-meta 备选面板
.\update-resources.ps1 -Only geo    # 仅地理数据
.\update-resources.ps1 -Only core   # 仅 Mihomo 内核
.\update-resources.ps1 -Only cfst   # 仅 CloudflareSpeedTest
.\update-resources.ps1 -Only nodes  # 免费节点聚合
python gen_config.py                # 重新生成配置
python node_aggregator.py           # 抓取免费节点
python node_aggregator.py --test    # 抓取+测速
python node_aggregator.py --merge clash-config.yaml  # 合并到配置
```
